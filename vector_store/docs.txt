Google Gemini is a family of large language models developed by Google. It can generate text, answer questions, and perform various natural language processing tasks.  Key points: - Gemini can be accessed via API for text generation. - It is used in this project to generate answers based on retrieved documents and user queries. 
---
RAG (Retrieval-Augmented Generation) is a technique that combines retrieval of relevant documents with generative AI models to answer questions more accurately.  Key points: - RAG retrieves relevant documents from a knowledge base. - It then uses a language model to generate an answer using both the question and the retrieved documents. - This approach is useful for question answering, chatbots, and knowledge assistants. 
---
Streamlit is an open-source Python library that makes it easy to create and share beautiful, custom web apps for machine learning and data science.  Key points: - Streamlit apps are written in pure Python. - It is used in this project to provide a web interface for the RAG application.
